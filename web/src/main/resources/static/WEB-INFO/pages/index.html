什么是AB测试平台？

Get Started Quickly

AB测试来源于生物学研究，基本思想是控制变量法。控制某些业务变量，将同一业务分成两个对照组，通过统计学方法分析采集来的业务数据，对比观察实际收益。

Valuable Platform

AB测试对数据驱动业务增长、功能改进和产品迭代等有着不可替代的重要作用。它可以有效提升产品质量，第一时间改进业务，嗅探到市场的微妙变化，帮助企业在激烈的市场竞争中取得先机。
